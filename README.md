# Kings Company Authentication suite

This repository contains the sign-up and sign-in suite for Kings Company, developed using Bootstrap, HTML, and CSS. These suite are designed with a modern, responsive layout to ensure an optimal user experience across various devices.

## Project Overview

The authentication suite serve as the gateway for users to access Kings Company's online services. The design focuses on usability, accessibility, and a clean, professional appearance, reflecting the brand's identity.

### Key Features

- **Responsive Design**: Built with Bootstrap to ensure seamless display on mobile, tablet, and desktop devices.
- **User-Centric Layout**: Easy-to-navigate forms with clear instructions to enhance user experience.
- **Clean and Modern Aesthetics**: CSS styles tailored to align with Kings Company's branding.
- **Cross-Browser Compatibility**: Tested on major web browsers to ensure consistent performance.

### Technologies Used

- **Bootstrap4**: For responsive grid layout and components.
- **HTML5**: Markup language for structuring the content.
- **CSS3**: Custom styles to complement the Bootstrap framework.

### Installation

To get a local copy up and running, follow these steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Dev-King001/kings-company-auth-suite.git
   ```

2. **Navigate to the Project Directory:**
   ```bash
   cd kings-company-auth-suite
   ```

3. **Open the Files:**
   Open `Sign-in.html` in your preferred browser to view the sign-up and sign-in suite.
